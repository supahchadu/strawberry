## IOS Application focused on version 9.x.x -> Upgrading when IOS 10.x.x launched
Demonstrates GPS Geolocation finding and use of real-time servers to manipulate data.

### Libraries used:
* Cocoapods 1.x.x
* Ruby 3.2
* XCode 7.3 -> Beta 4
* Firebase (database, storage, auth, testlabs)
* Geofire (geolocation)
* Pods
* Swift 3
* Object-C
* Facebook-SDK
